vu que nous approchons les evenements de Noel et Bonne année, je veux creer un app web pour permettre auz gens de faire des petites lettre pour souhaiter auz autres le nouvel an et le noel, pour creer une petite afiche de noel contenant son nom avec un decor de noelm, pour permettre de mettre une petite affiche des noms, des souhaits, ou ndes photos de couple, des cadeaux,...

# Architecture 

1. Page d'accueil : Présentation + options de création
2. Créateur d'affiches : Éditeur avec templates de Noël
3. Galerie publique : Voir les créations des autres
4. Partage : Liens uniques, téléchargement image, partage réseaux
5. Comptes utilisateurs (optionnel) : Sauvegarder ses créations


Idées de templates :
    - Lettre au Père Noël style
    - Carte de vœux numérique
    - Photo de couple avec cadre de Noël
    - Liste de cadeaux décorative
    - Compte à rebours du Nouvel An
